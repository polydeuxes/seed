# Question graph pass 089 — current-state graph artifact

## Coverage boundary

This artifact is a self-contained current-state graph for presently located and supported repository evidence. It covers supported nodes, directed relations, branches, refusal boundaries, lawful stops, mediated connections, disconnected shores, historical roads, and Unknown edges visible in the current Book, implementation, tests, reports, history, and app-visible surfaces. It is not a complete future architecture, universal ontology, mandatory pipeline, runtime owner map, planner, scheduler, lifecycle engine, or closure of preserved Unknowns.

## Legend

- `[C]` constitutional standing-changing relation.
- `[K]` constitutional constraint or refusal boundary.
- `[M]` methodological warrant exchange.
- `[I]` implementation-backed direct or mediated handoff.
- `[R]` correspondence or representation relation.
- `[H]` historical relation no longer live.
- `[U]` Relationship Unknown or disconnected expected neighbor.

## Full graph view

```text
operator/external expression
  [R] -> addressable pressure
  [K] -> stop/refusal/clarification when source, scope, authority, or provenance is insufficient

addressable pressure
  [C/M] -> possible inquiry through bounded translation
  [U] -/-> Inquiry Need (no current direct implementation bridge)

bounded goal + current position + repository/world uncertainty testimony
  [I] -> Inquiry Need

Inquiry Need
  [I] -> bounded inquiry frontier when one exact selected need and coherent boundary testimony are supplied
  [R/U] -> possible inquiry only through separate translation/admission evidence
  [K] -/-> formed question, inquiry opening, observation authorization, action, recording, mutation

possible inquiry
  [C] -> admitted inquiry movement through local admission authority
  [K] -> refusal, deferment, Unknown, lawful stop

admitted inquiry movement
  [R/I] -> individual bounded question where explicit bounded question artifact is produced
  [C] -> bounded examination eligibility where local evidence warrants
  [K] -/-> answer, execution, completion, mutation

typed Unknown
  [K] -> lawful question form, evidence posture, conclusion strength, permitted claims, forbidden inferences, next inquiry, stop
  [I/R] -> public generic unknown projection only as compatibility representation
  [K] -/-> QuestionFamily selector, generic unresolved text, resolution

Question Shape
  [M] -> Survey Warrant Family when recurrence/form evidence supports survey warrant
  [R/U] -> QuestionFamily only in explicitly registered cases
  [K] -/-> registry, router, public answerability, ontology

Survey Warrant Family
  [M] -> survey/investigation artifact warrant
  [K] -/-> QuestionFamily admission or bounded ask eligibility

QuestionFamily
  [I] -> exact-family lookup
  [I] -> eligibility
  [I] -> selected bounded work surface/value when permitted and required args are satisfied
  [I] -> dispatch or presentation handoff
  [R] -> answer responsibility and family-local answer surface
  [R/U] -> individual bounded question only through family-local contracts such as the constitutional-pipeline special case
  [K] -/-> Question Shape, Survey Warrant authority, semantic routing, free-text admission

individual bounded question
  [I] -> constitutional pipeline projection/selection/composition when already established
  [I] -> examination frontier with corpus and candidate work evidence
  [R] -> evidence/explanation/finding only through local examination or answer surface
  [K] -/-> fact, authority, family selection, ledger write, mutation

bounded inquiry frontier
  [K] -/-> question formulation, inquiry opening, source/observation selection, access authorization, execution, recording, mutation
  [U] -/-> individual bounded question formulation bridge absent

candidate examination work
  [I] -> examination frontier classification

examination frontier
  [I/M] -> method applicability
  [I/K] -> policy-constrained selection
  [K] -> lawful inactivity/no-selection/blocked/deferred/unsupported/unknown/conflict

method applicability + policy + selection
  [I] -> probe request when selected work, handoff, frontier, candidate set, and applicability match
  [K] -/-> dispatch, evidence gathering, execution

probe request
  [R/U] -> operational realization handoff only where a separate performer exists
  [K] -/-> scheduled or performed examination, provider trust, facts, mutation

performance/output
  [C] -> finding only through evidence/provenance/authority evaluation
  [K] -/-> understood, admitted knowledge, truth

finding
  [C/R] -> answer or bounded result when family-local answer responsibility or result sufficiency exists
  [K] -/-> established fact, sufficiency, completion by itself

answer responsibility
  [R/I] -> family-local answer
  [K] -/-> universal answer engine

answer
  [K/C] -> sufficiency evaluation
  [K] -/-> completion by itself

sufficiency
  [C] -> local completion or bounded result when criteria and authority exist
  [K] -/-> universal completion

completion
  [K/U] -/-> retirement unless separate retirement/deletion authority exists
  [R] -> egressable result where representation limits are preserved

bounded result
  [C/R] -> outward representation preserving limits

outward representation
  [C/R] -> delivery or availability
  [K] -/-> stronger truth, broader authority, receipt, understanding, reliance, mutation

delivery or availability
  [K/U] -/-> receipt, understanding, reliance absent separate proof

operator response
  [R/C] -> new pressure through ingress translation
  [K/U] -/-> correction, supersession, reopening, mutation of prior artifact absent separate authority

historical raw constitutional-pipeline field preparation
  [H] -/-> current pipeline ingress; deleted and superseded by already-established bounded-question contract
```
