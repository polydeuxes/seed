# Projection Authority and Provenance Audit

This audit is an implementation-backed inventory of projection-producing surfaces. It does not introduce ontology, redesign projections, reconcile concepts, or propose fixes.

## Classification key

| Classification | Meaning in this audit |
| --- | --- |
| Fact-backed | Directly supported by stored facts or evidence-preserving state rows. |
| Derived | Deterministic transformation of stored facts, evidence, or event-ledger state. |
| Candidate | Intentionally uncertain, heuristic, ambiguous, or possibility-oriented output. |
| Diagnostic | Visibility, debug, integrity, or coverage output; not repository knowledge. |
| Presentation | Display convenience, formatting, ranking, grouping, bounded selection, or summarization. |
| Cache | Persistence/performance artifact; not repository knowledge. |

## Authority summary

| Output group | Repository-knowledge status |
| --- | --- |
| Observed/current facts, evidence, observations, and event-derived State collections | Repository knowledge when projected from ledger events; facts/evidence retain direct support/provenance. |
| Inferred facts, fact support, aliases, catalog relationships, source navigation, evidence graph, confidence rows, filesystem rows/arithmetic | Derived repository knowledge when traceable to facts/evidence and deterministic catalogs/rules. |
| Shared storage candidates, storage topology ambiguities, cluster mount visibility groups, inquiry orientation related material | Candidate-only projection behavior; useful visibility but not assertions or topology/intent authority. |
| State Summary, storage shape/topology summaries, decision context selection, source/inquiry rendering, reachability tables | Presentation/read-model selection behavior; not repository knowledge. |
| Graph issues, conflicts, contradictions, integrity summary, classification coverage, capability inventory, reachability audit, timing/debug/cache status | Diagnostic-only visibility; not truth, correctness, health, or repository knowledge. |
| Projection snapshots, summary snapshots, derived-index snapshots | Cache/performance artifacts; removable and reconstructable from ledger/projected State. |
