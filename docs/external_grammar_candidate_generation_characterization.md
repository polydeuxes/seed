# External Grammar Candidate Generation Characterization

## Status

This note preserves a working methodology candidate. It is not implementation, constitutional law, or a recovered architectural boundary.

Repository authority wins wherever implementation evidence disagrees.

## Context

Recent discussion revisited the already recovered constitutional distinction:

```text
External Grammar

!=

Seed Grammar
```

The new observation is not the existence of that boundary. Instead, a simple sentence exposed a possible compression within the translation process.

Example:

```text
The cat jumped over the fence.
```

The sentence appears singular. However, multiple constitutionally valid artifact decompositions are compatible with the same external grammar.

Examples include:

```text
Entity

Action

Obstacle

Traversal

Outcome
```

or:

```text
Capability

Obstacle Traversal
```

or:

```text
Observed Movement Path
```

The external grammar alone does not determine which artifact graph should become Seed Grammar.

## Working Hypothesis

Current thinking suggests the process may be better characterized as:

```text
External Grammar

↓

Candidate Artifact Graphs

↓

Evidence

↓

Admitted Seed Grammar
```

rather than:

```text
External Grammar

↓

Seed Grammar
```

This resembles physiological patterns already independently recovered elsewhere in the architecture.

Examples include:

```text
Operator Communication

↓

Candidate Questions

↓

Evidence

↓

Lawful Continuation
```

and:

```text
Capability Signals

↓

Candidate Conclusions

↓

Evidence

↓

Lawful Continuation
```

The recurring observation is not that language is special. Rather, ambiguity itself may participate in the organism's existing candidate-generation physiology.

## Architectural Significance

If this characterization proves correct, external language is not directly translated into Seed Grammar. Instead, external language orients the organism toward one or more candidate artifact structures.

Evidence then determines which candidate, if any, may be admitted.

This would make language another instance of the organism's recurring physiological pattern rather than an exceptional subsystem.

## Important Distinction

This discussion does **not** reopen the previously recovered boundary:

```text
External Grammar

!=

Seed Grammar
```

That distinction is already established.

This characterization concerns the possible physiology that exists **between** those two artifacts.
