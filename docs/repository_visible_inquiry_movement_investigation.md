# Repository Visible Inquiry Movement Investigation

## Central finding

Repository evidence supports a distinction between repository-visible inquiry state and human-interpreted inquiry state.

The strongest observation is:

```text
humans currently observe more inquiry movement
than repository-visible surfaces
```

## Repository-visible inquiry artifacts

Strongest visible candidates:

```text
unknown
boundary
```

Partially visible:

```text
pressure
finding
```

Mostly document-visible:

```text
supported conclusion
unsupported conclusion
open question
```

## Inquiry movement visible today

Examples:

```text
selection rationale pressure
    -> selection_path

reference pressure
    -> reference_selection

projection pressure
    -> projection_shape
```

And:

```text
pressure
    -> implementation surface
```

These movements are observable from repository-visible surfaces.

## Inquiry movement requiring human interpretation

Examples:

```text
finding
    -> new question

supported conclusion
    -> next investigation

pressure
    -> transformed pressure

unknown
    -> investigation target
```

These are primarily reconstructed from investigation documents.

## Can Seed observe pressure transformation today?

Current evidence suggests:

```text
not directly
```

Seed can observe endpoints.

Humans currently reconstruct most:

```text
pressure A
    -> pressure B
```

relationships.

## Inquiry artifacts as observation candidates

Repository evidence supports a mixed interpretation.

Documentation-oriented:

```text

acceptance answers
evidence reviewed
```

Observation-like candidates:

```text
unknown
boundary
pressure
finding
```

These repeatedly behave as inquiry-state units rather than formatting.

## Counterexamples

Repository-visible movement:

```text
selection pressure
    -> selection_path

reference pressure
    -> reference_selection
```

Human-interpreted movement:

```text
finding
    -> new question

conclusion
    -> next investigation

pressure
    -> transformed pressure
```

## Supported conclusions

1. Repository-visible inquiry movement exists but is limited.
2. Humans currently observe substantially more inquiry movement than repository-visible surfaces.
3. Question-to-surface and pressure-to-implementation are the strongest repository-visible movement forms.
4. Finding-to-question, conclusion-to-next-investigation, and pressure-to-transformed-pressure remain primarily human interpreted.
5. Seed cannot directly observe pressure transformation today from repository-visible evidence alone.
6. Unknown, boundary, pressure, and finding are the strongest inquiry-artifact candidates for observation-like behavior.
7. A substantial boundary remains between repository-visible inquiry state and human-interpreted inquiry state.

## Acceptance answers

### What inquiry movement is already repository-visible?

Question-to-surface and pressure-to-implementation movement.

### What inquiry movement currently depends on human interpretation?

Finding-to-question, conclusion-to-next-investigation, unknown-to-investigation-target, and pressure-to-transformed-pressure.

### Can Seed observe pressure transformation today?

Not directly.

### Are inquiry artifacts merely documentation structure, or potential observation candidates?

Some are documentation structure. Unknown, boundary, pressure, and finding increasingly behave like observation candidates.

### What is the current boundary between repository-visible inquiry state and human-interpreted inquiry state?

Repository-visible state preserves endpoints. Human interpretation reconstructs most movement between them.
