# Constitutional Relationship Slice 002: Artifact-Carried Profile Boundary

Repository authority wins.

## Boundary

This slice recovers exactly one additional constitutional-local ownership boundary:

```text
A competent local artifact producer owns preservation of the bounded
constitutional profile that makes later local consumption inspectable.

A later constitutional consumer uses only the carried profile for its bounded
role and does not backfill producer authority.
```

The selected boundary is adjacent to Slice 001 because Slice 001 recovered the
warrant/reliance relation as:

```text
Warrant carries bounded entitlement-and-limit.
Reliance consumes within that entitlement-and-limit.
```

The next repository-supported responsibility is not a decomposition of warrant,
entitlement, limit, reliance, or ownership itself. The adjacent recurring
evidence instead exposes the local handoff boundary between a competent producer,
a bounded artifact or profile, and a bounded later consumer.

This slice does not introduce a Warrant Engine, Reliance Engine, Authority
Framework, Artifact Registry, Runtime Grammar, Workflow, Planner, Scheduler,
universal relationship model, runtime architecture, or redesign.

## Selected boundary

Selected boundary:

```text
Artifact-carried constitutional profile preservation.
```

Ownership boundary:

```text
The competent local artifact producer owns carrying the profile it is competent
to preserve: subject, support/provenance, role, authority boundary, negative
authority, Unknowns, confidence, stop conditions, and non-promotion limits where
repository evidence supports those fields.

The downstream consumer owns only bounded use of that carried profile inside its
local role.
```

The artifact or profile mediates the boundary only to the extent it preserves the
profile. It is not a universal artifact ontology and not an implementation
registry.

## Required questions

The previously compressed responsibilities were:

- carrying bounded entitlement-and-limit for lawful reliance;
- consuming within that entitlement-and-limit;
- preserving the profile that makes later reliance inspectable when a bounded
  artifact participates;
- distinguishing competent local production of a profile from later bounded
  consumer use of that profile;
- refusing consumer backfill of missing producer authority;
- preserving negative authority, Unknowns, confidence, stop, and non-promotion
  limits across the handoff.

Slice 001 recovered the warrant/reliance boundary. It did not recover the

### 2. Which constitutional-local ownership boundary became directly observable?

The directly observable boundary is:

```text
A competent local producer owns preservation of the artifact-carried
constitutional profile.
A later consumer owns bounded use of that carried profile only within its local
role.
```

This is smaller than an artifact framework and stronger than saying that
language, expression, report prose, diagnostic rows, or downstream desire can
supply reliance by themselves.

### 3. What constitutional producer owns the recovered responsibility?

The recovered producer is not a single global role.

The constitutional producer is the **competent local artifact producer** for the
bounded artifact at issue: for example, an investigation writer,
characterization writer, recovery-slice producer, diagnostic declaration helper,
input inspector, observation ingestor, projection builder, or other
repository-supported local producer when the local evidence supports that role.

The producer owns only the profile it is competent to preserve. It does not own a
universal artifact system, downstream consumption, promotion, mutation,
implementation truth, or all future uses.

### 4. What bounded constitutional artifact or profile carries the boundary, if any?

The boundary is carried by a **bounded artifact-carried constitutional profile**,
where supported by repository evidence.

The profile may include:

- bounded subject or content identity;
- evidence/support;
- provenance;
- role or consumer boundary;
- authority boundary;
- negative authority;
- preserved Unknowns;
- confidence or evidence-strength limit;
- stop conditions;
- non-promotion limits.

No universal artifact ontology is recovered. Whether this phrase should become
canonical vocabulary remains Unknown.

### 5. What constitutional consumer uses it?

The constitutional consumer is the **bounded downstream consumer** competent for
the local role.

Examples from repository evidence include later investigations, family completion
audits, diagnostic inventory and shape-audit consumers, fact/projection/context
consumers, explanation consumers, and operators reading a bounded report surface
where the surface preserves its limits.

The consumer uses only what the artifact-carried profile preserves and only for
the supported role. It does not create the missing profile, widen authority,
resolve Unknowns, inflate confidence, mutate cluster truth, or treat presentation
vocabulary as repository knowledge.
