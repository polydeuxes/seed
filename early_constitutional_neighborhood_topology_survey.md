## Comparison after independent survey

### Recurring streets

The recurring streets currently visible are:

1. **Null / uncommitted -> evidence-gated movement or lawful stop.**
2. **Null -> bounded preserved signal / external representation.**
3. **Preserved signal -> read-only orientation.**
4. **Preserved signal -> candidate attachment or observation only through local support.**
5. **Current lawful condition -> readiness review.**
6. **Evidence sufficiency assessment -> supported transition, handoff, stop, or continued non-promotion.**
7. **No preserved event history -> first lawful preserved event.**
8. **Preserved history -> projection/audit input.**
9. **Preserved history + projection/warrant/evidence discipline -> bounded current condition or biography-like explanation.**

### Recurring neighborhoods

The earliest constitutional neighborhood appears as overlapping bounded districts rather than one city-scale mechanism:

- **Null district:** non-commitment, evidence gate, stop legitimacy, boundary preservation.
- **Presence district:** preserved signal, bounded external representation, opaque material, inquiry-note presence.
- **Readiness district:** current lawful condition, warrant, eligibility, promotion boundary, selection/refusal, stop discipline.
- **Local transition district:** law-constrained local orderings, identity, evidence, assessment, support, handoff.
- **History district:** no preserved event history, first local lawful event, append-only preservation, provenance.
- **Projection / biography-adjacent district:** read-only inspection, projection, explanation, current lawful condition, unknown preservation.
- **Repository-orientation district:** read-only note orientation, deterministic related-material checks, absence/no-match visibility, richer-evidence need, stop.

### Recurring bridges

The strongest bridges are:

- **Evidence bridge:** the only lawful mover from Null toward supported condition.
- **Presence bridge:** bounded external representation can become present before interpretation.
- **Orientation bridge:** preserved material can be rendered read-only without mutation or truth creation.
- **Readiness bridge:** current lawful condition and evidence sufficiency connect proposed movement to continuation, refusal, or stop.
- **History bridge:** first lawful preserved event begins replayable preserved history.
- **Projection bridge:** preserved history becomes projection/audit input without becoming current truth by itself.
- **Biography bridge:** preserved history plus projection, warrant, evidence, confidence, contradiction, reliance, and unknown preservation can support later self-history explanation.

### Recurring neighborhood boundaries

- Null is distinct from absence, startup, initialization, unknown status, unavailable capability, and runtime activation.
- External representation is distinct from Observation, Evidence, Fact, attachment, interpretation, command, plan, recommendation, and inquiry graph.
- Readiness is distinct from promotion, permission, implementation readiness, mutation, execution, scheduling, and ownership.
- Local transition discipline is distinct from universal progression, workflow, pipeline, procedure, scheduler, planner, and implementation execution order.
- Preserved history is distinct from projection, cache, view, verification, objective reality, complete memory, and biography.
- Repository orientation is distinct from fact creation, semantic interpretation, next-step authority, mutation, and observer selection.
