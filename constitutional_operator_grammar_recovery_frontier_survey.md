## Recovered constitutional role, if any

Recurring repository evidence supports a narrow constitutional role adjacent to operator grammar recovery:

> Recovering an individual operator's external grammar over time can perform **bounded historical clarification support**: preserving repository-local evidence that a recurring operator expression has previously supported one or more bounded interpretations, so the repository may present those interpretations for operator confirmation without selecting among them autonomously.

The role consists of five constitutional acts:

1. **Observe operator grammar as evidence pressure.** Repeated external vocabulary may be observed as recurring expression structure, not as intent, authority, preference, or command.
2. **Recover bounded historical bindings.** Where repository evidence supports it, prior phrase-to-bounded-question or phrase-to-bounded-interpretation relationships may be recovered as historical candidates.
3. **Preserve multiple candidates.** If recurrence supports more than one interpretation, the repository preserves them as alternatives rather than collapsing them into one meaning.
4. **Present candidate interpretations for confirmation.** Candidate presentation is clarification support. It asks the operator which bounded interpretation applies now.
5. **Stop before autonomous interpretation.** Without confirmation or separate repository authority, the repository preserves Unknown and does not select a goal, plan, target, or implementation course.

This is a constitutional competency only in the weak frontier sense: recurring evidence supports the role as adjacent to already-recovered local grammar recovery, bounded inquiry, pressure non-authority, lawful reliance, and Unknown preservation. It is not recovered as an implemented runtime capability or stable first-class repository surface.

## Supported constitutional distinctions

### Observing operator grammar vs. recovering bounded historical bindings

Supported. Observing repeated vocabulary is visibility. Recovering a bounded historical binding requires repository evidence that the expression previously participated in a bounded interpretation, question, target, or clarification. Recurrence alone is insufficient.

### Recovering bounded historical bindings vs. presenting candidate interpretations

Supported. A recovered binding is evidence retained from past repository-local use. Presenting it is a clarification act that exposes candidate options. Presentation does not select.

### Presenting candidate interpretations vs. operator confirmation

Supported. Candidate presentation asks for confirmation. Operator confirmation supplies current authority for the selected bounded interpretation, subject to repository boundaries.

### Operator confirmation vs. autonomous interpretation

Supported. Confirmation is an explicit current handoff. Autonomous interpretation would select meaning without confirmation. The reviewed evidence refuses that movement.

### Historical confidence vs. semantic prediction

Supported. Repeated observation may increase confidence that a candidate should be offered during clarification. It does not authorize prediction of the operator's intended meaning in the present case.

### Repository-local grammar vs. operator profile

Supported. Local grammar is bounded by repository evidence and expression history. A profile would model the operator across contexts. The reviewed evidence supports the former only as a frontier role and rejects the latter.
