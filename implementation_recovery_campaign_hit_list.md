## Audit method

Each candidate is audited only for:

- constitutional stability: constitutional confidence and implementation stability;
- implementation locality: whether the implementation boundary is already sufficiently local that one bounded implementation slice is likely to recover additional knowledge, plus local implementation owner and confidence;
- expected knowledge gain;
- dependency pressure;
- campaign disposition.

Campaign dispositions are limited to:

- `Immediate candidate`
- `Strong candidate`
- `Wait for additional implementation evidence`
